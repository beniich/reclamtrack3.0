Inference Engine

Interface for executing low-latency machine learning predictions across our global edge network. Version 2.4.0-stable.

Production Ready
v2.4 LTS
POST /v2/predict/{model_id}

Execute Prediction

Submits raw input data to a specific model instance for inference. Supports batch processing up to 128 records.

Parameters

model_id required

The unique identifier for the trained model version.

string (uuid)
X-API-KEY required

Authentication token passed via header.

header

Request Body

application/json 
{
  "inputs": [
    {
      "data": [0.24, 0.55, 0.81],
      "metadata": { "tag": "prod_test" }
    }
  ],
  "config": {
    "confidence_threshold": 0.85
  }
}
