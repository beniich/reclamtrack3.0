AUTHENTIFICATION (Auth API)

Générez des jetons d'accès pour une communication API sécurisée. Nécessite une Clé API et un Secret.

import requests

url = "https://api.aidata.ai/v1/auth/token"
payload = {
    "api_key": "VOTRE_CLÉ_API",
    "api_secret": "VOTRE_SECRET_API"
}

response = requests.post(url, json=payload)

print(response.json())
